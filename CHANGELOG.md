# Changelog

Все заметные изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [0.1.0] — 2026-07-14

Первый стабильный релиз Telegram-бота для учёта личных расходов.

### Добавлено

- Разбор суммы, описания и даты из текстовых сообщений на русском языке.
- Поддержка дробных сумм с точкой или запятой и расходов «вчера».
- Локальная классификация категорий `products`, `entertainment`, `medicine`, `other` с подтверждением при неоднозначности.
- Карточка подтверждения расхода с возможностью изменить категорию или отменить запись.
- Команды дневных и месячных отчётов, просмотра/удаления последнего расхода, категорий и CSV-экспорта.
- Автоматическая месячная рассылка в последний день месяца с учётом `TIMEZONE` и защитой от повторной отправки.
- Ограничение доступа по `ALLOWED_TELEGRAM_USER_IDS`.
- SQLite по умолчанию и поддержка PostgreSQL через `DATABASE_URL`.
- SQLAlchemy-модели, миграции Alembic, структурированные логи и graceful shutdown.
- Контейнеризация Docker Compose и интерактивный установщик для Ubuntu/Debian.
- Автоматические тесты для парсинга, категорий, отчётов, часовых поясов, доступа и удаления расходов.
